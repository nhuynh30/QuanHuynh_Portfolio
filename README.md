# QuanHuynh Portfolio

This repository contains a React + Vite portfolio website for a rising CS junior at George Mason University seeking an internship.

## Files

- `index.html` — Vite entrypoint
- `src/main.jsx` — application bootstrap
- `src/App.jsx` — portfolio content and structure
- `src/styles.css` — responsive styling and layout
- `package.json` — project dependencies and scripts

## Run locally

1. Install dependencies:

```bash
cd /Users/quanhuynh/Desktop/Portfolio
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Customize the site

- Replace the placeholder email in `src/App.jsx` with your own.
- Add or update project descriptions in `src/App.jsx`.
- Add resume download, GitHub links, or more sections.
- Update the copy to match your internship goals and experience.
